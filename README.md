# webengineer-subject-finalproject-microservice-express
This microservice verifies the soil's suitability for growing date palms.

# Tech Stack
<ul>
  <li>Node</li>
  <li>NPM</li>
  <li>Express</li>
  <li>JavaScript</li>
</ul>

# Run project
Open a CLI in root folder of project and then run:

```sh
npm install
node index.js
```
